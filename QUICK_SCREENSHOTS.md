# التقاط لقطات الشاشة - دليل سريع ⚡

## الخطوات (10 دقائق فقط!)

### 1️⃣ شغل التطبيق
```bash
python src/main.py
```

### 2️⃣ التقط 6 صور أساسية

#### الصورة 1: الشاشة الرئيسية
- انتظر حتى تظهر المواقيت
- اضغط `Windows + Shift + S`
- حدد نافذة التطبيق
- احفظ باسم: `01_main.png`

#### الصورة 2: نافذة الأذان
- اضغط زر الإعدادات ⚙️
- غير وقت الفجر لـ 1 دقيقة من الآن
- انتظر حتى تظهر نافذة الأذان
- التقط الصورة: `02_adhan.png`

#### الصورة 3: الإعدادات - عام
- اضغط زر الإعدادات ⚙️
- التقط صورة التبويب الأول
- احفظ: `03_settings.png`

#### الصورة 4: الإعدادات - الأذان
- اختر تبويب "الأذان"
- التقط الصورة: `04_adhan_settings.png`

#### الصورة 5: الإعدادات - الموقع
- اختر تبويب "الموقع"
- التقط الصورة: `05_location.png`

#### الصورة 6: الوضع الداكن
- فعّل "الوضع الداكن"
- ارجع للشاشة الرئيسية
- التقط الصورة: `06_dark.png`

### 3️⃣ احفظ الصور
```
d:\تطبيق المؤذن\screenshots\
```

## ✅ تم!

الآن عندك 6 صور جاهزة للمتجر!

**التحقق من الدقة:**
```powershell
cd screenshots
Get-ChildItem *.png | ForEach-Object {
    $img = [System.Drawing.Image]::FromFile($_.FullName)
    Write-Host "$($_.Name): $($img.Width)x$($img.Height)"
    $img.Dispose()
}
```

يجب أن تكون جميعها 1366x768 أو أعلى ✅
